My Services

What we can do for you

Application Penetration Test

I'll perform a manual penetration test for your application supported by automated tools and scripts to assess the targets in-scope.

Which includes but not limited to the following:
- Access Controls and Authorization Issues
- Input Validation & Data Protection
- Error Handling
- Functionality Abuse / Business Logic Flaws
- Files and resources
- Web Services
- Application Misconfigurations

Network Penetration Test

I'll perform a manual penetration test for your organization's network and look for the weaknesses in it's security posture.

Which includes but not limited to the following:
- Insecure Services
- Missing Patches
- System Misconfigurations
- Active Directory Weaknesses
- Network ACLs
- Network Segmentation
- Use of default passwords

Build and Configuration Review

I will perform a build review to inline with the CIS Benchmark standard for the following servers:

- AWS
- Azure
- Windows Server
- Windows Machine
